# User Management Dashboard

A responsive React web application to view, add, edit, delete, search, sort, filter, and paginate user details using JSONPlaceholder mock API.

## Features

- View users
- Add new user
- Edit existing user
- Delete user
- Search users by name, email, or department
- Sort users by table columns
- Filter users by first name, last name, email, and department
- Pagination with 10, 25, 50, and 100 limits
- Client-side form validation
- API error handling
- Responsive user interface

## Tech Stack

- React
- Vite
- JavaScript
- Axios
- CSS
- JSONPlaceholder API
